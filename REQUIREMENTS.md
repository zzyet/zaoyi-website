---

## 1. 技术栈

| 需求 | 实现 |
|------|------|
| 使用 React 技术 | React 19 + Vite 脚手架 |
| 动画效果 | Framer Motion（滚动渐显、hover 动效）+ Canvas 粒子网络背景 |
| 不要黑色背景 | 亮色主题：白色 `#ffffff` + 浅灰 `#f7f8fc` 交替分区 |
| 亮色主题 | CSS 变量驱动，完整亮色设计令牌体系 |

## 4. 背景动画（神经网络粒子系统）

| 迭代 | 需求 | 实现 |
|------|------|------|
| v1 | 炫酷科技感、神经网络动画 | 25 个神经元节点 + 100 背景粒子 + 连线 + 数据脉冲流 + 科技网格叠加 + 扫描线 |
| v2 | 修复粒子聚团问题 | 删除向心力、增加节点互斥力（80px）、网格均匀分布初始化、加强阻尼、弹性边界 |
| v3 | 鼠标互动 | 神经元 200px 内被吸引、连线靠近鼠标加亮（2.5×）、节点光晕放大至 5× |

## 5. UI 设计系统（Stripe × Apple × Framer 融合）

| 设计维度 | 方案 | 参考来源 |
|----------|------|----------|
| 字体 | Source Sans 3（300/400/600），等宽 Source Code Pro | Stripe |
| 标题字重 | 300（轻量优雅） | Stripe |
| 标题颜色 | `#061b31`（深海军蓝） | Stripe |
| 正文颜色 | `#64748d`（石板灰） | Stripe |
| 强调色 | `#533afd`（Stripe 紫），单一紫色做所有交互 | Apple |
| 阴影系统 | 蓝调多层 `rgba(50,50,93,...)` + `rgba(0,0,0,...)` | Stripe |
| 圆角 | 4-8px（保守精密） | Stripe |
| 分区背景 | 白色 ⇄ 浅灰 `#f7f8fc` 交替 | Apple |
| 动效 | 神经网络粒子 + 数据脉冲 + 卡片光晕呼吸动画 | Framer |
